Daily Mood Tracker

Project Description
The Daily Mood Tracker is a simple web-based mini project that helps users record and track their daily moods. Users can select their mood, add notes, and monitor emotional patterns over time using a clean and interactive interface.

Features

* Record daily mood
* Add notes for each day
* View mood history
* Daily and monthly mood tracking
* Simple and user-friendly design

Technologies Used

* HTML
* CSS
* JavaScript

Purpose
This project helps users understand their emotional habits and maintain self-awareness by tracking moods regularly.
Conclusion
The Daily Mood Tracker provides an easy way to monitor emotions and encourages users to build a healthy habit of self-reflection.
